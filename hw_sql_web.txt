1.Реалізувати функцію видалення задач по кліку по кнопці x.
Для цього треба:
    Створити DeleteServlet, в якому перевизначити метод doGet
    В методі doGet отримати параметр запиту id (задачі), який передати в метод delete класа TodoListDao
    Після видалення задачи перенаправити запит на головну сторінку
    В класі TodoListDao імплементувати метод delete()

2.В продовження практичного завдання реалізувати можливість відмічати завання викоанними.
Для цього потрібно:
    2.1. Додавти сервлет UpdateServlet
    2.2. Прив'язвати цей сервлет до url /complete
    2.3. Обробити запит методом GET при цьому отримати id задачі, для якої потрібно змінити статус
    2.4. Знайти в базі даних задачу за вказаним id (з використанням DAO)
    2.5. Змінити статус задачі на звротній (якщо завершена - перевести в не завершена і навпаки)
    2.6. Перенаправити запит на головну сторінку
    2.7. Перевірити на працездатність
    2.8. Робочий код відправити на перевірку через mystat
